# Produktion

Auf den vorgelagerten Hauptprozess «Produktion» wird nicht weiter eingegangen, da dieser sehr spezifisch für jeden Datensatz ist. Bei OGD handelt es sich immer um eine Sekundärnutzung von Daten, die bereits angefallen sind. Eine Erhebung von Daten für OGD ist nicht vorgesehen.
